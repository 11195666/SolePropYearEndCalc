# 个体户税费计算器

一个微信小程序，用于计算个体工商户年度经营所得的应缴税费，帮助个体户快速了解增值税、附加税、个人所得税的构成，方便做税务规划和年度汇算。

## 功能

- **增值税计算** — 支持 0%（免税）、1%（优惠）、3%（一般纳税人）三档税率
- **附加税费计算** — 含城建税、教育费附加、地方教育附加，自动应用六税两费减半政策
- **个人所得税计算** — 经营所得 5%-35% 超额累进税率，自动适用年所得 200 万以下减半征收优惠
- **专项附加扣除** — 支持住房、子女教育、赡养老人、继续教育、大病医疗五项扣除填报
- **社保公积金扣除** — 支持灵活就业养老/医疗、个人公积金等年缴纳额抵扣
- **详细计算过程** — 点击各项税费可查看完整计算步骤

## 使用方法

1. 克隆项目，用微信开发者工具打开
2. 在 `project.config.json` 中将 `appid` 替换为你自己的小程序 AppID
3. 编译运行即可
